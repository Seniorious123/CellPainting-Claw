# Segmentation

The segmentation branch standardizes mask export, single-cell crop generation, and preview generation.

It is designed to preserve the validated backend segmentation assets while exposing a cleaner orchestration layer for:

- full segmentation runs
- mask-export delivery suites
- single-cell crop generation
- downstream DeepProfiler export preparation
