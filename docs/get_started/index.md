# Get Started

The fastest way to understand the project is to validate the installation, inspect the resolved configuration, and run one standardized entrypoint.

## Recommended First Steps

1. Create the environment and install the package.
2. Run the smoke test from the first-run guide.
3. Inspect the example configuration.
4. Run the main end-to-end pipeline entrypoint.
5. Inspect the skill catalog and public Python API.
