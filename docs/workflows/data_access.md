# Data Access

The data-access layer is responsible for discovering and planning upstream assets before workflow execution.

It includes support for:

- Cell Painting Gallery prefix and dataset/source discovery
- reusable download-plan generation
- Quilt package discovery
- cpgdata-backed inventory and index access
